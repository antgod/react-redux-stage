## react-redux组件脚手架

#### 使用方法：
+ 全局安装脚手架命令
```
$npm i ant-redux -g
```
+ 创建项目
```
$ant-redux init
$#输入项目目录xxx
```

+ 新增模板
```
$ant-redux add
```

+ 运行项目
```
$cd xxx
$npm start
# 浏览器会自动打开页面
```

+ 编写项目
直接编写xxx下composite下面组件与mock数据即可。相当简单
